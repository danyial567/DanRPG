# DanRPG - دنیای استانک

🎮 **یک بازی RPG 3D موبایلی در دنیای استانک**

## 📖 داستان
شما **دان** هستید - فرزند پادشاه شیاطین دنیا. در یک دنیای تاریک و پر از خطر، باید شهرت و قدرت خود را کسب کنید و سلطنت را فتح کنید.

## 🎯 ویژگی‌ها
- 🌍 **دنیای باز 3D** - آزادانه کاوش کنید
- ⚔️ **سیستم مبارزه** - با دشمنان شیاطین درگیر شوید
- 💪 **سیستم کاراکتر** - مهارت‌ها، تجربه و سطح‌بندی
- 🎮 **کنترل موبایلی** - جوی‌استیک و دکمه‌های سهل‌الاستفاده
- 🏆 **سیستم پیشرفت** - ماموریت‌ها و چالش‌ها
- 💰 **اقتصاد بازی** - سکه و اشیاء جمع‌آوری کنید

## 🛠️ تکنولوژی
- **Engine**: Unity 2022 LTS
- **زبان**: C#
- **پلتفرم**: iOS & Android

## 🚀 شروع سریع
1. **Unity نصب کنید** (2022 LTS یا بالاتر)
2. **Repository را clone کنید**
3. **پروژه را در Unity باز کنید**
4. **Scene** > **MainScene** را اجرا کنید

## 🎮 کنترل‌ها
- **Joystick چپ**: حرکت کاراکتر
- **دکمه راست**: حمله
- **دکمه بالا**: استفاده از مهارت
- **دکمه منو**: باز/بستن منو

## 📱 مربوط به موبایل
- پشتیبانی **لمسی کامل**
- تطابق خودکار **صفحه**
- بهینه‌سازی برای **باتری**
