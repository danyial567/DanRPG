# 🎮 کد‌های تقلب و دیباگ DanRPG

## 🎯 کد‌های تقلب اصلی

### فعال‌سازی تقلب‌ها
```
Ctrl + Shift + C
```

وقتی تقلب‌ها فعال شوند، می‌تونید از کد‌های زیر استفاده کنید:

### کد‌های موجود

| کد | توضیح |
|---|---|
| **H** | درمان کامل کاراکتر |
| **G** | اضافه کردن 1000 سکه |
| **L** | بالا رفتن یک سطح |
| **E** | اضافه کردن 500 تجربه |
| **M** | پر کردن مانا کاملاً |
| **K** | کشتن تمام دشمنان |
| **S** | افزایش سرعت حرکت |
| **T** | نمایش آمار کاراکتر |
| **D** | فعال‌سازی حالت بی‌آسیب |

---

## 🐛 کنسول دیباگ

### باز کردن کنسول
```
Alt + D
```

### دستورات کنسول

```
help          - نمایش تمام دستورات
stats         - نمایش آمار کاراکتر
heal          - درمان کامل
gold [عدد]    - اضافه کردن سکه
level [عدد]   - تنظیم سطح
```

### مثال‌ها

```
gold 5000           → اضافه کردن 5000 سکه
level 10            → تنظیم سطح به 10
heal                → درمان کامل
```

## 📝 نحوه استفاده

### 1. فعال کردن تقلب‌ها
1. فشار دکمه: `Ctrl + Shift + C`
2. پیام موفقیت در Console Unity ظاهر می‌شود

### 2. استفاده از کد‌های تقلب
بعد از فعال‌سازی، فقط کلید مورد نظر را فشار دهید:
```
H → درمان
G → سکه
L → سطح
```

### 3. استفاده از کنسول دیباگ
1. `Alt + D` را فشار دهید
2. دستور را بنویسید
3. دکمه "اجرا" را بزنید

---

## 🔒 نکات امنیتی

### برای انتشار نهایی
تقلب‌ها را **غیرفعال** کنید:

```csharp
#if UNITY_EDITOR
    // فقط در حالت توسعه
#else
    // تقلب‌ها غیرفعال
#endif
```

### یا
از متغیر `cheatsEnabled` استفاده کنید و آن را در تنظیمات کنترل کنید.

## 🎮 مثال استفاده

### سناریو 1: تست مبارزه
```
1. Ctrl + Shift + C  (فعال کردن تقلب)
2. K                 (کشتن تمام دشمنان)
3. G                 (اضافه کردن سکه)
```

### سناریو 2: تست سطح‌بندی
```
1. Alt + D           (باز کردن کنسول)
2. level 50          (تنظیم سطح)
3. stats             (نمایش آمار جدید)
```

### سناریو 3: تست عملکرد
```
1. Ctrl + Shift + C
2. S                 (افزایش سرعت)
3. K                 (کشتن دشمنان برای تست)
```

## 🛠️ افزودن کد‌های جدید

برای افزودن کد تقلب جدید، `CheatManager.cs` را ویرایش کنید:

```csharp
// در CheckCheatCodes()
if (Input.GetKeyDown(KeyCode.YOUR_KEY))
    YourCheatFunction();

// سپس تابع را اضافه کنید
private void YourCheatFunction()
{
    Debug.Log("کد تقلب اجرا شد!");
    // کد شما
}
```
